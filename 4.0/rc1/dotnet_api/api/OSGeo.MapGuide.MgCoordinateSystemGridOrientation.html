Class MgCoordinateSystemGridOrientation

Namespace
OSGeo.MapGuide
Assembly
OSGeo.MapGuide.Geometry.dll

An enumeration of the supported values for the m_Orientation member of several objects related to grids and graticules. This value is used to qualify objects which are of the "iso" type. I.e. a grid line is referred to as an isoline as it is the locus of points which have a specific value for either the easting or the northing. It is this value which indicates which. Thus, a grid line which is classified as having an "EastWest" orientation will be a isoline which is the locus of points which share a common easting value, and the "m_Value" element of that object will be an easting value. Note that in this example, the line is typically a vertical line.
