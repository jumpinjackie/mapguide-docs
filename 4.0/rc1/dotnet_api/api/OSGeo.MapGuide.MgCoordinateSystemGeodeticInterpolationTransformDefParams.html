Class MgCoordinateSystemGeodeticInterpolationTransformDefParams

This class holds all parameters that are specific to grid-file based transformation. That is, where the actual datum shift value for a coordinate is determined by interpolating between grid point values as defined in the respective grid file(s).

GetGridFiles()

Returns an ordered, non-live collection MgCoordinateSystemGeodeticTransformGridFile elements currently used by this parameter object. Changing the collection does NOT have an affect on this parent parameter instance. Instead, any API client has to modify the collection and then set it back via SetGridFiles. The list returned can be an empty but will never be NULL.
