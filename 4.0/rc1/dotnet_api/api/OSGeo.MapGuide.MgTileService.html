GetTile(MgResourceIdentifier, string, int, int, int)

Returns the specified base map tile for the given map or tile set. If a cached tile image exists it will return it, otherwise the tile is rendered and added to the cache.

public MgByteReader GetTile(MgResourceIdentifier resource, string baseMapLayerGroupName, int tileColumn, int tileRow, int scaleIndex)

Parameters

resource MgResourceIdentifier

Input Resource identifier for the map definition or Tile Set Definition

baseMapLayerGroupName string

Input Specifies the name of the baseMapLayerGroup for which to render the tile.

tileColumn int

Input Specifies the column index of the tile to return.

tileRow int

Input Specifies the row index of the tile to return.

scaleIndex int

Input Scale index to render. Most detailed scale is index 0.
